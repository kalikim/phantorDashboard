# PHANTOR DASHBOARD

Phantor dashboard is a interactive visualization for PHANTOR 

## Installation

Inside the Phantor dashboard folder run the following 

```bash
pip install -r requirements.txt
```

## Usage


```bash
python main.py
```
Access the map by running the browser
```bash
localhost:8000
```
